## 1. 프로젝트 개요

### 1.1 프로젝트명
**카톡 코치 AI** — 카카오톡 대화 파일로 관계 코칭하는 온디바이스 AI 앱

### 1.2 콘셉트
카카오톡 대화 파일을 분석해서 AI가 상대방의 감정/관심도를 분석하고, 적절한 답장 조언을 제공하는 모바일 앱

### 1.3 핵심 가치 제안
- **온디바이스 AI**: 모든 데이터가 사용자 기기 내에서 처리 (서버 전송 없음)
- **한국 특화**: 카카오톡 + 한국어 대화 맥락 특화
- **프라이버시 퍼스트**: "내 대화가 외부로 나가지 않는다"

### 1.4 대상 사용자
- 데이트 중이거나 연애 초보인 20~30대
- 카카오톡으로 상대와 대화하며 답장에 고민하는 사람
- AI 기반 관계 코칭에 관심 있는 사람

## 4. 상세 기술 설계

### 4.1 시스템 아키텍처

```
┌─────────────────────────────────────────────────────────────┐
│                        사용자 단말                          │
│  ┌─────────────┐  ┌──────────────┐  ┌─────────────────┐   │
│  │ 카카오톡 파일 │  │  온디바이스 AI │  │  로컬 스토리지  │   │
│  │  (.txt)     │  │ (Gemma 2B Q4)│  │   (MMKV)       │   │
│  └──────┬──────┘  └──────┬───────┘  └────────┬────────┘   │
│         │                  │                   │             │
│         └──────────────────┼───────────────────┘             │
│                            │                               │
│                     ┌───────┴───────┐                       │
│                     │ React Native │                       │
│                     │   앱 (Expo)   │                       │
│                     └───────────────┘                       │
└─────────────────────────────────────────────────────────────┘
```

**핵심: 모든 처리가 사용자 기기 내에서 완료. 서버 없음.**

### 4.2 기술 스택

| 레이어 | 선택 | 이유 |
|--------|------|------|
| **앱 프레임워크** | React Native (Expo SDK 52) | iOS/Android 동시 개발 |
| **AI 모델** | Gemma 2B Q4 / Llama 3.2 1B Q4 | 온디바이스 최적화, GGUF 형식 |
| **카카오톡 파싱** | 자체 파서 (정규식) | 카카오톡 내보내기 파일 직접 파싱 |
| **UI 컴포넌트** | React Native Paper (MD3) | Material Design 3 |
| **상태 관리** | Zustand v5 | 가볍고 React Native 친화적 |
| **로컬 스토리지** | MMKV + AsyncStorage | 빠른 읽기/쓰기, 암호화 |
| **구독 관리** | RevenueCat | 앱스토어/플레이스토어 통합 |
| **On-device AI** | Transformers.js + @huggingface/jinja | GGUF 양자화 모델 구동 |

### 4.3 AI 최적화 전략

#### 양자화 (Quantization)
| 포맷 | 크기 | 성능 유지율 | 추천 사용 |
|------|------|------------|----------|
| Gemma 2B Q4 | ~1.3GB | 90% | **모바일 최적화** |
| Llama 3.2 1B Q4 | ~0.7GB | 85% | 가벼운 사용 |

#### 프롬프트 엔지니어링
```javascript
// 한국 상담 특화 프롬프트
const SYSTEM_PROMPT = `
당신은 20년 경력의 커플 상담사입니다.
한국인의 감정 표현과 대화 스타일을 깊이 이해하고 있습니다.

분석时请注意以下几点:
1. 한국어의 반말/존댓말 구분
2. 이모티콘의 감정 표현 의미
3. 한국의 데이트 문화 맥락
`;
```
