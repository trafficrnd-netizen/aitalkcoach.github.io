# AI톡 코치 - SYSTEM_ALERT_WINDOW 권한 선언서

## 📋 Play Store 정책 준수 선언

### 1. 사용하는 권한
- **SYSTEM_ALERT_WINDOW** (android.permission.SYSTEM_ALERT_WINDOW)

### 2. 권한 사용 목적
카카오톡 대화 분석을 위한 **플로팅 버블(AI 답변 추천 버튼)** 기능 제공

### 3. 구체적인 사용 시나리오

```
[시나리오]
1. 사용자가 카카오톡에서 대화 내용을 복사
2. AI톡 코치의 플로팅 버튼이 클립보드에서 카카오톡 대화 패턴을 감지
3. 사용자가 플로팅 버튼을 탭
4. 대화 분석 모달이 표시됨
5. AI가 감정/관심도/답장 추천을 분석
6. 사용자가 추천 답장을 복사 → 카카오톡에 붙여넣기
```

### 4. Google 정책 준수 사항 ✅

#### User-Initiated Requirement (사용자 조작 요구)
- ✅ **사용자가 반드시 플로팅 버튼을 탭해야 표시**
  - 앱 설치 후 첫 실행 시 "활성화하기" 안내 모달 표시
  - 사용자가 직접 "활성화하기" 버튼을 눌러야 기능 활성화
  - 비활성화 옵션: 설정 또는 플로팅 메뉴에서 언제든 해제 가능

#### Transparency Requirement (투명성 요구)
- ✅ **앱 아이콘과 이름이 항상 표시**
  - 플로팅 버튼 상단에 "AI톡코치" 배지 항상 표시
  - 버튼에 💬 아이콘과 앱 색상(Indigo) 사용
- ✅ **어떤 앱 위에 표시되는지 명확히 구분**
  - 반투명한 오버레이가 아닌, 독립적인 버튼 형태
  - 카카오톡 UI를 가리지 않음

#### Revocability Requirement (해제 가능성)
- ✅ **설정에서 비활성화 가능**
  - 앱 설정 화면에서 "플로팅 버튼 비활성화" 옵션
  - 플로팅 버튼 롱프레스 → 메뉴에서 "비활성화" 선택
  - Android 권한 설정에서도 해제 가능

### 5. 다른 앱 위에 표시되는 방식

| 항목 | 내용 |
|------|------|
| 표시 형태 | 56x56px 원형 버튼 |
| 위치 | 화면 가장자리 (드래그 가능) |
| 색상 | Indigo (#6366F1) + 흰색 테두리 |
| 아이콘 | 💬 (말풍선 이모지) |
| 앱 이름 표시 | 버튼 상단에 "AI톡코치" 배지 |

### 6. 주요 기능 설명 (Play Console 입력용)

```
플로팅 AI 버튼 기능:

AI톡 코치는 카카오톡 대화 위에 표시되는 
AI 답변 추천 버튼을 제공합니다.

[활성화 방법]
1. 앱에서 플로팅 버튼을 활성화
2. 카카오톡 대화 내용을 복사
3. AI 버튼을 탭하여 분석 시작

[주요 특징]
• 사용자가 직접 탭해야 표시
• 항상 앱 이름/아이콘 표시
• 카카오톡 대화를 외부로 전송하지 않음
• 모든 분석은 기기 내에서 완료
• 언제든 설정에서 해제 가능

이 기능은 SYSTEM_ALERT_WINDOW 권한을 사용하며,
Google Play 정책의 모든 요구사항을 충족합니다.
```

### 7.スクリーンショット (Play Console용)

플로팅 버튼이 표시된 상태의スクリーンショット 필요:

1. **활성화 안내 모달** - 기능 설명 + 활성화 버튼
2. **플로팅 버튼 표시 상태** - 카카오톡 위에 표시된 버튼
3. **분석 모달** - 버튼 탭 후 표시되는 분석 화면
4. **설정 해제 옵션** - 설정 또는 메뉴에서 해제 방법
