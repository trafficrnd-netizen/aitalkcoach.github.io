AI톡 코치

최종 업데이트: 2024년 1월 1일

1. 수집하는 개인정보

AI톡 코치(이하 "앱")는 사용자의 프라이버시를高度重视합니다. 앱은 사용자의 카카오톡 대화 파일을 분석하여 관계 코칭 서비스를 제공합니다.

📱 수집하는 정보

  • 카카오톡 대화 파일: 사용자가 직접 선택하여 업로드한 대화 내보내기 파일(.txt)
  • 앱 사용 정보: 일일 분석 횟수, 구독 상태 (기기 내 저장)
  • 기기 정보: 앱 정상运作 확인을 위한 최소한의 정보

💡 핵심 원칙: 사용자가 직접 선택한 파일만 처리됩니다. 자동으로 데이터를 수집하지 않습니다.

2. 개인정보 처리 목적

  • 카카오톡 대화 분석 및 감정/관계 코칭 서비스 제공
  • 맞춤형 답장 추천 생성
  • 앱 기능 개선 및用户体验优化
  • 구독 서비스 관리

3. 개인정보 처리 방식

🔥 온디바이스(기기 내) 처리

모든 대화 분석은 사용자의 기기 내에서만 이루어집니다. 사용자의 카카오톡 대화 파일이 외부 서버로 전송되지 않습니다. 앱이 인터넷 연결 없이도 기본 기능을 사용할 수 있도록 설계되어 있습니다.

☁️ 서버 연동 (선택적)

  • 구독 관리: RevenueCat을 통한 구독 상태 확인
  • 앱 업데이트: 앱스토어 연동을 통한 업데이트 알림

4. 데이터 저장 및 삭제

저장 위치

  • 기기 내: 분석 기록은 사용자의 기기에만 저장
  • 외부 전송 없음: 대화 파일이 서버로 전송되지 않음

데이터 삭제 방법

설정 → 데이터 관리 → 모든 데이터 삭제를 통해 언제든지 삭제할 수 있습니다.

⚠️ 참고: 앱을 삭제하면 모든 로컬 데이터가 자동으로 삭제됩니다. 구독 관리는 각 플랫폼(Apple/Google)의 구독 설정을 통해 관리됩니다.

5. 개인정보 공유

앱은 사용자의 개인정보를 제3자와 공유하지 않습니다. 단, 다음의 경우を除きます:

  • 법령에 의한 요구가 있는 경우
  • 사용자가 명시적으로同意한 경우

6. 접근권한 요청 (Android)

📂 저장소 접근 (READ_EXTERNAL_STORAGE)

사용 이유: 사용자가 선택한 카카오톡 대화 파일(.txt)을 읽어들이기 위해 필요합니다.

파일 선택 창을 통해 사용자가 직접 파일을 선택하는 경우에만 접근됩니다.

🖥️ 다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW)

사용 이유: 카카오톡 앱 위에서 AI 답변 추천 버튼(플로팅 버블)을 표시하기 위해 필요합니다.

  • 사용자가 먼저 "활성화하기"를 탭해야 표시됩니다
  • 항상 최상단에 표시되지 않고, 앱 아이콘/이름이 명확히 표시됩니다
  • 설정 또는 플로팅 버튼 메뉴에서 언제든 비활성화할 수 있습니다

7. 플로팅 버블 기능

플로팅 버블은 Google Play의 " 다른 앱 위에 표시 " 정책에 따라 구현되었습니다:

  • ✅ 사용자가 반드시 조작(탭)해야 활성화됩니다
  • ✅ 앱 아이콘과 이름이 항상 표시됩니다
  • ✅ 사용자의 대화가 아닌, 클립보드 내용만 확인합니다
  • ✅ 언제든 설정에서 비활성화할 수 있습니다
  • ✅ 네트워크 연결 없이도 작동합니다

8. 아동의 개인정보

앱은 만 14세 미만 아동의 서비스를 제공하지 않습니다. 만약 만 14세 미만임이 확인되면, 해당 데이터는 즉시 삭제됩니다.

9. 개인정보 처리방침 변경

개인정보 처리방침은 법령更改 또는 서비스 개선을 위해更新될 수 있습니다. 변경 시 앱内通知 및 이 페이지에 공지됩니다.

10. 문의하기

개인정보 처리와 관련하여 문의가 있으시면, 다음으로 연락주세요:

  • 이메일: support@aitalkcoach.app
  • 앱 내: 설정 → 문의하기

11. 권리 행사

사용자는 다음의 권리를 행사할 수 있습니다:

  • 개인정보 접근 및 열람 요청
  • 개인정보更正 요청
  • 개인정보 삭제 요청
  • 개인정보 처리 제한 요청
  • 삭제 요청: 설정 → 데이터 관리에서 직접 삭제 가능
