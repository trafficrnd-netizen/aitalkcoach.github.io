## 📝 긴 설명 (최대 4,000자)

### 🎯 한 줄 소개
카카오톡 대화 파일을 분석해서 **감정, 관심도, 관계 상태**를 파악하고, 상황에 맞는 **AI 답장 추천**까지 받아보세요!

---

### 💡 주요 기능

**📊 대화 심층 분석**
- 카카오톡 대화 내보내기 파일로 분석
- 감정 흐름 파악 (긍정/부정/중립)
- 상대방 관심도 측정
- 관계 발전 단계 분석

**💬 상황별 답장 추천**
- 데이트 제안에 맞는 답장
- 감사 인사에 어울리는 반응
- 미묘한 감정への返事
- 상황에 맞는 톤 추천

**🎈 실시간 AI 버튼 (플로팅 버블)**
- 카카오톡 위에 AI 버튼 표시
- 대화 복사 → 버튼 탭 → 분석 완료
- 카카오톡을 벗어날 필요 없이 바로 답장

---

### 🔒 프라이버시 & 권한

**모든 분석은 기기 내에서 완료됩니다**

**필요한 권한과 그 이유:**

📂 **저장소 접근 (READ_EXTERNAL_STORAGE)**
→ 사용자가 선택한 카카오톡 대화 파일(.txt)을 읽기 위해
→ 파일 선택 창을 통해 사용자가 직접 파일을 선택하는 경우에만 접근

🖥️ **다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW)**
→ 카카오톡 위에 AI 답변 추천 버튼(플로팅 버블)을 표시하기 위해
→ 사용자가 먼저 "활성화하기"를 탭해야 표시됩니다
→ 설정 또는 플로팅 버튼 메뉴에서 언제든 비활성화 가능

**우리의 약속:**
- ✅ 대화 파일이 외부 서버로 전송되지 않음
- ✅ 인터넷 연결 없이도 기본 분석 가능
- ✅ 사용자가 직접 선택한 파일만 처리
- ✅ 언제든지 모든 데이터 삭제 가능

---

### 👥 이런 분들에게 추천

- 💑 커플과의 대화를 더 잘 이해하고 싶은 분
- 💼 업무 관계의 메시지를 더 잘 해석하고 싶은 분
- 🤔 상대방이 나에게 어떤 감정을 느끼는지 알고 싶은 분
- ✨ 상황에 맞는 완벽한 답장을 찾고 싶은 분

### 📋 사용 방법

1️⃣ 카카오톡 → 대화방 → 설정 → 대화 → 대화 내보내기
2️⃣ 파일 형식: **텍스트** 선택 후 저장
3️⃣ AI톡 코치에서 파일 선택 → 분석 시작
4️⃣ 감정 분석, 관심도, 추천 답장 확인

**또는 플로팅 버튼 모드:**
1️⃣ 앱에서 플로팅 버튼 활성화
2️⃣ 카카오톡 대화 복사
3️⃣ 💬 버튼 탭 → 바로 분석!

### ❓ 자주 묻는 질문

**Q. 대화가 외부로 전송되나요?**
A. 아니요. 모든 분석은 사용자의 기기 내에서만 이루어지며, 데이터가 외부로 전송되지 않습니다.

**Q. 어떤 카카오톡 파일을 지원하나요?**
A. 카카오톡 내보내기 → 파일 형식 '텍스트'로 저장한 .txt 파일을 지원합니다.

**Q. 플로팅 버튼이 자꾸 표시되나요?**
A. 아닙니다. 사용자가 직접 "활성화하기"를 탭해야 표시되며, 언제든 설정에서 해제할 수 있습니다.
